Overview

Gets version of the low-level Virgil cryptographic library. This version is different from the version of the VirgilFoundation pod itself.

In general major and minor versions for the low-level library and VirgilFoundation should be the same, but it is not always the case.

This is utility class which might come in handy when some bugs are observed and it is crucial to know exact versions of every single component used in the application.
